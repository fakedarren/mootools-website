each Loop through a collection of Elements

Example:

<ul class="list">
	<li>One</li>
	<li>Two</li>
	<li>Three</li>
</ul>

$$('.list li').each(function(element, index){
	// Alerts 'Item 1: One', 'Item 2: Two', 'Item 3: Three'
	alert('Item ' + index + ': ' + element.get('text'));
});

Syntax:

$$(selector).each(fn, bind);
fn Required
(function) the function to run on each element. 
$$(selector).each(function(element, index, collection){
	// your code
});

This function gets passed:

element
(element) the current element.
index
(number) the index in the collection.
collection
(elements) the original collection of elements.
bind Optional
(object) The object to use as 'this' within the function.
