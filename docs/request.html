Request Our Ajax wrapper

The MooTools Ajax base class. If you only need to do something simple, then you may be able to use some of the simpler methods available.

Syntax

var myRequest = new Request(options);   // create the Request object
myRequest.send();                       // send the request

Show Options:

url
(string) the url to request
method
(string) The HTTP method for the request, can be either 'post' or 'get'. The default is 'post'.
data
(element, object or string) the data to send as part of the request.

You can send a (single) element:

data: $('myForm')
// If using a CSS selector, you need to select a single element, like so:
data: document.getElement('#myForm')
// You can't use $$ as that returns an *array* of elements
data: $$('#myForm') // doesn't work

You can use an object:

data: {
	value1: 'foo',
	value2: 'bar'
}

Or you can pass a string.

data: 'value1=foo&value2=bar'
onSuccess
(function) An event that fires when the request is successful. 
onSuccess: function(responseText, responseXML){
	// code goes here
}

Example:

new Request({
	url: 'submit.php',
	data: $('myForm'),
	onSuccess: function(response){
		alert(response);
	}
}).send();
Alternatives to Request

  • If you want to load HTML into an element easily, check out Element.load.

    <div id="foo"></div>

$('foo').load('/content.php');

  • If you want to send a form easily, check out Element.send.

    <form id="foo" method="post" action="submit.php">
	<input type="text" name="bar" value="1" />
</form>

$('foo').send();
