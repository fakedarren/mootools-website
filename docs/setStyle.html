setStyle Set a CSS style

Sets a single CSS property. Need to set more than one property? You need setStyles

Examples:

$$('.list li').setStyle('color', 'red');

$$('input[type=text]').setStyle('border', '1px solid #94CC25');

// all these are the same
$('container').setStyle('height', '100px');     // by ID, single element
$$('#container').setStyle('height', '100px');   // by CSS selector
$$('#container').setStyle('height', 100);       // by CSS selector, pass a number

Syntax:

$(id).setStyle(property, value);
$$(selector).setStyle(property, value);
property Required
(string) the property to set.
value Required
(string or number) the value to set the property to.
