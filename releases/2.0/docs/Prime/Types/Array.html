Array

These methods are only provided as a shim for browsers that do not support them natively.

  • Array.forEach

    Loop through an array and perform a function on each item.

  • Array.indexOf

    Returns the index of the first item in the array equal to the specified value.

  • Array.every

    Checks to see if every item in an array passes the passed function.

  • Array.some

    Checks to see if some items in an array pass the passed function.

  • Array.filter

    Create a new array, filtered using the passed function.

  • Array.map

    Create a new array of values passed through the passed function.

  • Array.install

    Install the Array methods on the native prototype.
