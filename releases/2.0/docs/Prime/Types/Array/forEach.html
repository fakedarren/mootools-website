Array.forEach Loop through an array and perform a function on each item

Example:

_([1, 2, 3]).forEach(function(item){
	console.log(item);
});


// Namespaced method
(function(array){
	array.forEach([1, 2, 3], function(item){
		console.log(item);
	});	
})(MooTools.array);


// Or if you can extend the native prototype
MooTools.array.install();

[1, 2, 3].forEach(function(item){
	console.log(item);
});

Syntax:

_(myArray).forEach(fn, bind);
fn Required
(function) the function to run on each item. 
_(myArray).forEach(function(item, index, array{
	// Your code here
})

This function gets passed:

item
(mixed) the current item.
index
(number) the index in the array.
array
(array) the original array.
bind Optional
(object) The object to use as 'this' within the function.
(comments)
